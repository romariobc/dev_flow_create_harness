# Testes/evals reutilizáveis

Coloque aqui scripts e templates de avaliação que são genéricos o suficiente para servir a múltiplos projetos (ex: um runner que carrega um dataset do LangSmith e aplica um critério de similaridade semântica). Casos de teste específicos de um projeto vão em `projetos/<projeto>/evals/`, não aqui.

Ainda vazio — popule conforme o primeiro projeto real precisar de um runner de avaliação reaproveitável.
